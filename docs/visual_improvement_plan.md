## 📋 상세 개선 항목

### 1. 애니메이션 시스템 (우선순위: 높음)

#### 1-1. 빵 생산 애니메이션

```
현재: 버튼 클릭 → 즉시 완료
개선: 버튼 클릭 → 진행바 → 완료 이펙트
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 오븐 슬롯 | 회전/깜빡임 애니메이션 | ⭐ |
| 진행바 | 부드러운 채워짐 | ⭐ |
| 완료 이펙트 | 반짝임 + 스케일 업 | ⭐⭐ |
| 연기 이펙트 | 오븐에서 피어오르는 연기 | ⭐⭐⭐ |

#### 1-2. 판매 애니메이션

```
현재: 빵 사라짐 → 골드 증가
개선: 빵 플로팅 → 코인 이펙트 → 골드 증가
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 빵 플로팅 | 위로 떠오르며 페이드아웃 | ⭐⭐ |
| 골드 플라이 | 코인이 HUD로 날아감 | ⭐⭐⭐ |
| 팝업 숫자 | +100G 등 텍스트 표시 | ⭐ |

#### 1-3. UI 버튼 피드백

```
현재: 클릭 → 즉시 반응
개선: 클릭 → 스케일 → 사운드 → 액션
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 스케일 트위닝 | 눌림 → 복귀 (0.1초) | ⭐ |
| 호버 효과 | 커서 올리면 밝아짐 | ⭐ |
| 사운드 | 클릭 사운드 재생 | ⭐ |

---

### 2. 캐릭터 애니메이션 (우선순위: 중간)

#### 2-1. 요정 idle 애니메이션

```
현재: 정지 이미지
개선: 살짝 움직임 (호흡, 깜빡임)
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 호흡 | Y축 살짝 이동 (sin 파) | ⭐ |
| 깜빡임 | 눈 깜빡이기 (랜덤) | ⭐⭐ |
| 파티클 | 요정 주변 빛나는 입자 | ⭐⭐⭐ |

#### 2-2. 요정 고용 애니메이션

```
현재: 버튼 비활성화
개선: 요정 나타남 → 반짝임 → 자리 잡음
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 등장 연출 | 페이드인 + 스케일 | ⭐⭐ |
| 반짝임 | 별 파티클 | ⭐⭐ |
| 이동 | 제자리로 날아감 | ⭐⭐⭐ |

---

### 3. 이펙트 시스템 (우선순위: 중간)

#### 3-1. 레벨업 이펙트

```
현재: 텍스트만 변경
개선: 화면 플래시 → 폭죽 → "LEVEL UP!" 텍스트
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 화면 플래시 | 흰색 페이드 | ⭐ |
| 폭죽 파티클 | 여러 색 입자 폭발 | ⭐⭐⭐ |
| 텍스트 팝업 | 크게 나타나며 페이드아웃 | ⭐⭐ |

#### 3-2. 업그레이드 이펙트

```
현재: 숫자만 변경
개선: 글로우 → 파티클 → 색상 변화
```

| 항목 | 내용 | 난이도 |
|------|------|--------|
| 글로우 | 버튼 주변 빛나기 | ⭐⭐ |
| 파티클 | 위로 솟구치는 입자 | ⭐⭐ |
| 색상 변화 | 강조 색상으로 변경 | ⭐ |

## 🛠️ 기술 스택

### 애니메이션

- **Tween**: 간단한 트위닝 (위치, 스케일, 색상)
- **AnimationPlayer**: 복잡한 시퀀스
- **Shader**: 글로우, 그라데이션 효과

### 파티클

- **GPUParticles2D**: 성능 좋은 파티클
- **CPUParticles2D**: 간단한 이펙트

### 사운드

- **AudioStreamPlayer**: 2D 사운드
- **AudioBus**: 볼륨 관리
