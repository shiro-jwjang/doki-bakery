# 🎮 Game Design Document

## 두근두근 베이커리 (Doki-Doki Bakery)

### Version 1.0 | 프로토타입 기준 | 2026-03-01

---

## 📌 Executive Summary

**두근두근 베이커리**는 마법의 요정들이 함께하는 픽셀아트 방치형 경영 시뮬레이션 게임입니다. 플레이어는 작은 베이커리를 시작으로 다양한 빵을 굽고 판매하며, 귀여운 요정 파티시에를 고용하고 베이커리를 성장시킵니다.

### 핵심 가치

- **힐링**: 따뜻한 분위기의 코지한 베이커리 경영
- **성장**: 방치형 시스템으로 지속적인 보상
- **수집**: 다양한 빵, 요정, 업적 수집
- **접근성**: 간단한 조작, 누구나 즐길 수 있는 캐주얼 게임

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| **게임명** | 두근두근 베이커리 (Doki-Doki Bakery) |
| **장르** | 픽셀아트 방치형 경영 시뮬레이션 (Idle Tycoon) |

### 1.2 유니크 셀링 포인트 (USP)

1. **요정 파티시에 시스템**: 귀여운 요정들이 생산을 도와주는 독특한 컨셉
2. **계절 콘텐츠**: 봄/여름/가을/겨울 한정 빵으로 지속적인 신선함
3. **힐링 아트 스타일**: 16bit 픽셀아트의 따뜻한 파스텔 톤
4. **TDD 개발**: 안정적인 코드베이스와 지속 가능한 개발

### 1.3 타겟 오디언스

| 세그먼트 | 설명 |
|---------|------|
| **주 타겟** | 20-40대 캐주얼 게이머, 방치형 게임 경험자 |
| **부 타겟** | 힐링/코지 게임 팬, 수집 욕구가 강한 플레이어 |

## 2. 핵심 게임 루프

### 2.1 메인 루프

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   빵 선택 → 오븐 제작 → 완료 알림 → 진열대 → 판매 → 골드 획득  │
│       ↓                                         ↓           │
│   요정 고용 (생산 가속)                    업그레이드 구매     │
│       ↓                                         ↓           │
│   새로운 빵 해금 ← ← ← ← ← ← ← ← ← ← ← ← 레벨 업             │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 세션 루프 (3-5분)

1. **접속**: 오프라인 보상 수령
2. **생산**: 빵 선택 → 오븐에 굽기
3. **판매**: 완성된 빵 진열대 확인
4. **성장**: 업그레이드 구매 / 요정 고용
5. **종료**: 다음 접속 예정 시간 확인

### 2.3 장기 루프 (일/주 단위)

- **일일**: 출석 보상, 일일 퀘스트 완료
- **주간**: 새로운 빵 해금, 요정 레벨업
- **월간**: 시즌 콘텐츠, 마일스톤 달성

## 3. 게임 메카닉

### 3.1 생산 시스템

#### 3.1.1 오븐 슬롯

| 항목 | 내용 |
|------|------|
| **기본 슬롯** | 2개 |
| **최대 슬롯** | 5개 |
| **확장 방법** | 업그레이드 구매 (진열대 확장) |

#### 3.1.2 생산 시간 계산

```
생산 시간 = BaseTime × (1 - FairyLevel × 0.05) × (1 - UpgradeLevel × 0.1)
최소 생산 시간 = BaseTime × 0.2 (20%)
```

**예시 (크루아상, BaseTime 10초)**:

- 기본: 10초
- 요정 Lv5: 10 × (1 - 0.25) = 7.5초
- 업그레이드 Lv3: 7.5 × (1 - 0.3) = 5.25초
- 최소: 10 × 0.2 = 2초

#### 3.1.3 빵 카테고리

| 카테고리 | 설명 | 예시 |
|---------|------|------|
| **basic** | 기본 빵 | 식빵, 크루아상 |
| **pastry** | 페이스트리 | 마들렌, 스콘 |
| **cake** | 케이크 | 딸기 케이크, 초코 케이크 |
| **special** | 특수 빵 | 마카롱, 티라미수 |
| **seasonal** | 계절 빵 | 벚꽃 케이크, 눈사람 쿠키 |

### 3.2 판매 시스템

#### 3.2.1 판매 가격 계산

```
판매 가격 = IngredientCost × 2.5 + BasePrice × (1 + PlayerLevel × 0.1)
```

**예시 (식빵, IngredientCost 10, BasePrice 5, PlayerLevel 5)**:

- 가격 = 10 × 2.5 + 5 × (1 + 0.5) = 25 + 7.5 = 32.5G

#### 3.2.2 진열대 시스템

| 항목 | 내용 |
|------|------|
| **기본 슬롯** | 3개 |
| **최대 슬롯** | 10개 |
| **자동 판매** | 진열대에 있는 빵은 5초마다 1개씩 자동 판매 |

### 3.3 성장 시스템

#### 3.3.1 레벨 시스템

```
필요 경험치 = 100 × (1.5 ^ (Level - 1))
```

| 레벨 | 필요 경험치 | 누적 경험치 | 보상 |
|------|-----------|-----------|------|
| 1 | 100 | 100 | - |
| 2 | 150 | 250 | 크루아상 해금 |
| 3 | 225 | 475 | 딸기 요정 고용 가능 |
| 5 | 506 | 1,381 | 딸기 케이크 해금 |
| 10 | 3,844 | 9,849 | 특수 빵 카테고리 해금 |

#### 3.3.2 마일스톤

| 레벨 | 보상 |
|------|------|
| 5 | 두 번째 오븐 슬롯 해금 |
| 10 | 세 번째 오븐 슬롯 해금 |
| 15 | 특수 빵 카테고리 해금 |
| 20 | 네 번째 오븐 슬롯 해금 |
| 30 | 다섯 번째 오븐 슬롯 해금 |

### 3.4 오프라인 보상

```
오프라인 보상 = 기본 생산량 × 0.5 × 오프라인 시간
최대 보상 시간 = 24시간
```

**예시 (시간당 100G 생산, 8시간 오프라인)**:

- 보상 = 100 × 0.5 × 8 = 400G

#### 7.1.2 색상 팔레트

| 용도 | 색상 | HEX |
