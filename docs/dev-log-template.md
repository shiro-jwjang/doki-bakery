## 📝 작성 가이드라인

### 1. 작성 시점

- **매주 일요일 저녁** 또는 **월요일 오전** 작성 권장
- 한 주 작업을 마무리하고 정리하는 시간

### 2. 필수 포함 항목

- ✅ 최소 1개 이상의 완료 항목
- 📸 최소 1개 이상의 스크린샷/GIF
- 🗓️ 다음 주 계획

### 3. 톤앤매너

- 친근하고 긍정적인 톤
- 기술적 내용도 쉽게 설명
- 실패/어려움도 솔직하게 공유 (학습 포인트)

### 4. 해시태그 (SNS용)

- 필수: #개발일지 #인디게임
- 선택: #픽셀아트 #방치형게임 #코지게임

### 5. 미디어 첨크 팁

- 스크린샷: 새 기능, UI 변경사항 위주
- GIF: 동작이 중요한 기능 (애니메이션, 인터랙션)
- 비교 이미지: Before/After 효과적
